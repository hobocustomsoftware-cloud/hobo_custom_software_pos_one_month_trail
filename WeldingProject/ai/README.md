# AI Module — POS နဲ့ ကိုက်ညီသော AI

## လုပ်ဆောင်ချက်များ

1. **ဒါလေးရောမလိုဘူးလား (Upsell suggestions)**  
   Cart ထဲက ပစ္စည်းတွေအပေါ် အခြေခံပြီး အကြံပြုချက်။ Sales စာမျက်နှာမှာ cart ထည့်လိုက်ရင် အလိုအလျောက် ပြမယ်။

2. **အရောင်းရဆုံး စာရင်း**  
   API: `GET /api/ai/best-sellers/`။ မေးမြန်းရန် ထဲမှာ "အရောင်းရဆုံးစာရင်းပြပါ" လို့ မေးရင်လည်း ပြမယ်။

3. **ဈေးကွက်မှာ အကောင်းဆုံးဖြစ်အောင်**  
   "ဈေးကွက်မှာ အကောင်းဆုံးဖြစ်အောင် ဘယ်လိုလုပ်သင့်သလဲ" စသည့် မေးခွန်းကို စတိုးဒေတာ (အရောင်းရဆုံး၊ ယနေ့အရောင်း) နဲ့ ပေါင်းပြီး AI က အကြံပြုမယ်။

## API Endpoints

| Method | URL | ရည်ရွယ်ချက် |
|--------|-----|----------------|
| GET | `/api/ai/best-sellers/?limit=15` | အရောင်းရဆုံး ပစ္စည်းစာရင်း |
| POST | `/api/ai/suggest/` | body: `product_ids`, `product_names` — ဒါလေးရောမလိုဘူးလား အကြံပြုချက် |
| POST | `/api/ai/ask/` | body: `question` — မေးခွန်းမေးမြန်းခြင်း (အရောင်းရဆုံး / ဈေးကွက် အကြံပြု) |

အားလုံး **IsAuthenticated** လိုအပ်ပါတယ်။

## Environment (AI ချိတ်ဆက်ရန်)

| Variable | ရှင်းလင်းချက် |
|----------|------------------|
| **OPENAI_API_KEY** | OpenAI API key (OpenAI သုံးမယ်ဆိုရင်) |
| **AI_API_URL** | OpenAI-compatible endpoint။ မထည့်ရင် default က `https://api.openai.com/v1/chat/completions`။ Ollama သုံးမယ်ဆိုရင် `http://localhost:11434/v1/chat/completions` ထည့်ပါ။ |
| **AI_MODEL** | Model name (default: `gpt-4o-mini`)။ Ollama ဆိုရင် ဥပမာ `llama3.2` |

API key / URL မထည့်ထားရင်:
- **best-sellers** က ဒေတာပဲ ပြန်မယ် (အမြဲ အလုပ်လုပ်မယ်)။
- **suggest** က အရောင်းရဆုံး စာရင်းကနေ ရွေးပြီး အကြံပြုမယ် (AI မပါဘဲ)။
- **ask** က "အရောင်းရဆုံးစာရင်းပြပါ" မေးရင် စာရင်းပဲ ပြမယ်၊ အခြား မေးခွန်းတွေကို "AI မချိတ်ဆက်ရသေးပါ" ပြမယ်။
