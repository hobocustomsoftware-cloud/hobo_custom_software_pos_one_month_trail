# Simplified Universal Myanmar POS (Loyverse Style)

## 1. Fast Payment Integration (No Transaction ID)

- **Payment Methods:** Large 80px buttons for [ငွေသား], [KPay], [WavePay], [AYAPay], [Card/MPU].
- **Fast Flow:** Clicking any Mobile Pay button completes the sale immediately. No ID input needed.
- **Reporting:** Backend records the specific payment method for daily sales reports.

## 2. Dynamic Feature Toggle (Contextual UI)

- **General Settings:** Add toggles to Enable/Disable [Tax], [Discount], [Loyalty Points].
- **Clean UI:** If a feature is disabled, hide it completely from the POS screen to maintain simplicity.

## 3. Universal Industry Engine (25px Myanmar UI)

- **Phone/Solar/PC:** Serial Number (အမှတ်စဉ်) Tracking + Bundle/Set Management.
- **Pharmacy/Hardware:** 3-Level Units (လုံး/ကဒ်/ဗူး) or (ခု/ဒါဇင်/ဖာ) with auto-stock deduction.
- **Solar/Air-con:** Site Survey (ကွင်းဆင်းစစ်ဆေးခြင်း) stage with photo/measurement storage.

## 4. Senior-Friendly Design Standard

- **Typography:** 25px Noto Sans Myanmar.
- **Buttons:** 80px height throughout the app.
- **Layout:** Loyverse 70/30 Split. Large Green "ငွေချေမည်" button.
- **Visuals:** High contrast, no glassmorphism, pure white/light gray backgrounds.

## 5. Printing & Hardware

- **Receipt:** 58mm/80mm Thermal Print with Myanmar Unicode support.
- **Scanner:** Always-on global barcode listener.
